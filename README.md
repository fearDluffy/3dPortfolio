📥 How to Clone & Run Locally
# Clone the repo
git clone https://github.com/yourusername/3d-portfolio.git

# Enter the project directory
cd 3d-portfolio

# Install dependencies
npm install

# Start the dev server
npm run dev


Open http://localhost:5173
 to view locally.
